# Tower Playbooks

This directory contains a collection of Ansible playbooks curated for use with Ansible Tower (AWX). Each set of playbooks automates various administrative tasks across different systems and platforms, helping to streamline IT operations.

## 📁 Structure

- `activedirectory_playbooks/` – Manage Active Directory objects and settings.
- `azureplaybooks/` – Automate Azure resources and configurations.
- `cisco_playbooks/` – Configure Cisco network devices.
- `collections/` – Supporting Ansible collections required for the playbooks.
- `disable-ieesc/` – Scripts to disable Internet Explorer Enhanced Security Configuration.
- `idrac_playbooks/` – Automate Dell iDRAC setup and management.
- `inventory/` – Sample inventory files for use with Tower.
- `linux_playbooks/` – Playbooks for Linux server management.
- `vmware_playbooks/` – Automate tasks on VMware infrastructure.
- `windows_playbooks/` – Windows system administration via Ansible.
