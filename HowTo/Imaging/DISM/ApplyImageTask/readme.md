# Title: Apply-Image Files

---

## Purpose: In this directory you will find different versions of the Apply-Image.bat file. 

---

1. Apply-ImageV1
    - Orignal File, modifications included to answer the questions for no prompts
2. Apply-ImageV2 
    - Addtional logic to pull the machine model and apply device drivers based on model
3. Apply-ImageV3 
    - Addtional logic to prompt for company name based on folder list and copy unattend file to the panther directory
4. Apply-ImageV4 
    - Added logic to copy Unttendfile and SetupComplete.cmd
5. Apply-ImageV5 
    - Added functionality to prompt for device name to be stored and later used by SetupComplete.cmd to set device name
